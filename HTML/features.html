数据库&自动化

使用数据库
将工作翻译保存并加载为数据库。
支持多种数据库类型。
包括用于多种游戏和语言的主数据库。
直接从以前翻译的mod导入mod数据库。
支持创建自己的数据库。
适用于任何类型数据库的多个数据库编辑器。
实现自动翻译
通过Google和Yandex等数个在线工具翻译MOD
通过数个自动工具来翻译简单的翻译模式
通过数个自动翻译工具来解决特定的翻译模式

通用工具

MOD工具
在选项卡中打开multiple mods。
Automatic session saver.
Several comparison tools.
TEXT TOOLS
Spell-check for several languages in original and translated fields.
Text encoding support for many languages.
Several log options to track and revert changes in modified lines.
Search & replace text.
Several case edition tools.
SCRIPTING TOOLS
Scripts translation support.
Full scripts editor with syntax highlighting.
MEDIA TOOLS
Support for book translation.
Open NifScope models for specific lines.
Texture viewer for specific lines.
Media player for audio files linked to lines.
ACCESIBILITY TOOLS
Keyboard shortcuts for almost every utility.
Filters for lines by group, text and color.
Customizable line colors by translation type.
SPECIFIC TOOLS
FO4 mod conversion to "New dialog" mod.
Open, preview and extract several file formats in BSA/BA2 archives as a treeview. (FO4 support is partial)
TEAM TOOLS
Partial saves for work division..
Line coloring marks for team indications.
Text comments by line for team indications.
INTERFACE TOOLS
Customize fonts for several parts of the program.
Several display/adjust panels and buttons.
