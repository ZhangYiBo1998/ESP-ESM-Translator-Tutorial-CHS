引言

该程序是可移植的，只需要解压缩文件夹中的ZIP文件，然后启动ESP Translator.exe即可运行。 在此之后，必须手动下载和安装用于翻译游戏和语言的数据库，在某些情况下，还必须手动安装用于拼写检查的附加词典。


此外，程序第一次启动时，可能需要一些额外的手动调整，例如程序的界面语言和活动数据库的选择。 这些选项是在刚开始游戏时由弹出的消息窗口向用户询问的，但也可以稍后在“选项”窗口中进行调整。


这些数据库和词典可以从以下链接下载：

  • 主要数据库 : 它们不包括在程序中，必须从Nexus的作者mod页面上的可选文件部分下载。
  • Hunspell词典 : Hunspell词典包含在多种语言中，但如果您的语言不包含在内，请从链接中下载，并将.aff和.dic提取到程序的Dictionaries文件夹中。 如果默认情况下没有为您的语言安装同义词库.dat文件，也可以添加这些文件来获取您语言中的同义词建议。
安装

要求: 该程序需要安装Microsoft.NET framework 4.5才能正常运行。因此，它只与Windows Vista或更高版本兼容。它会自动检测系统的x32或x64体系结构，并自行配置。


1. 在任意位置解压缩应用程序ZIP文件。 这个程序是可移植的。

2. 将所选游戏的BDD数据库复制到安装文件夹中的BDD文件夹中。 在此之前必须从Nexus上的 ESP翻译器 页面下载。

3. 执行“ESP Translator.exe”。

4. 从列表中选择应用程序语言(Application Language)。 这可以稍后在选项/通用选项/应用程序语言(Options/General Options/Application Language)中更改。

5. 为要翻译的语言选择一个主数据库(Main Database)。 稍后可以在选项/主数据库/数据库(Options/Main Databases/DataBase)>中对此进行更改。

6. 打开“选项”窗口(Options Window)，然后选择“脚本和MCM(Scripts and MCM)”选项卡。

7. 在以下三种设置中选择翻译语言：
  • MCM分析“默认语言翻译MCM文件”
  • MCM翻译“默认语言翻译MCM文件”
  • 文本翻译“默认语言翻译文本文件”

可选的： 如果要使用不可用的语言进行拼写检查，请阅读上面引言部分的Hunspell词典段落。

更新

该程序可以通过覆盖旧版本文件来更新到新版本，但此过程有时会失败，最好单独安装，然后从以前安装的版本手动复制数据库和词典。通常，要复制的文件夹和文件包括：


文件夹

+ BDD: 此文件夹包含游戏的默认数据库，以及您创建的新数据库（Main、Custom、OnlyText…）。 如果更新版本的主数据库上传到mod页面，请先复制旧版本文件夹，然后用下载的覆盖它们。通过这种方式，您可以保留自己的数据库并更新默认数据库。

+ Bases Persos: 此文件夹包含个人保存的数据库，这些数据库是您保存活动翻译时创建的。

+ Dictionaries: 此文件夹包含拼写检查词典和同义词词库。如果你安装了自己的，你应该复制它。

+ Trans Autos: 此文件夹包含自动翻译模式，包括默认示例和您自己创建的模式。


文件

+ ESP Translator.settings: 这些文件包含选项(Options)窗口的全局设置以及菜单中的其他可调整选项。如果你更改了它们，你应该复制它。

+ NewDialogConfig.eet: 它应该将您的设置存储在Fallout 4-Convert for NewDialog Mod工具中。因此，您可能也有兴趣恢复它。


注释： ESP Translator.settings文件未包含在程序的压缩存档中，因此在更新时不会丢失以前的个人设置，以防您更喜欢安装新版本而不是旧版本。但是，如上所述，这种方法可能偶尔会失败。
