INTRODUCTION

The program is portable and just requires to uncompress the ZIP file in a folder and then launch ESP Translator.exe to be operative. After this, the databases for the games and language to translate, and, in some cases, additional dictionaries for spell check, must be downloaded and installed manually.

Also, the first time the program is launched, it may require some additional manual tweaks, such as the interface language fof the program, and the selection of the active databases. These options are asked to the user by emergent MessageBox windows while starting the game at first, but can also be tweaked later in the Options window.

These databases and dictionaries can be downloaded from the links below:
  • Main Databases: They are not included in the program, and must be downloaded from the author mod page at Nexus, under the Optional Files section.
  • Hunspell Dictionaries: Some of them are included for several languages, but if yours is not included, dowload it from the link, and extract the .aff and .dic into the Dictionaries folder of the program. The thesaurus .dat files can also be added to get synonyms suggestions in your language, if they are not installed by default for your idiom.
INSTALL

REQUIREMENTS: The program requires to have installed Microsoft .NET framework 4.5 to function properly. Hence, it is only compatible with Windows Vista or superior versions. It detects automatically for the x32 or x64 architecture of your system and configures itself .

1. Uncompress the Application ZIP file anywhere. The program is portable.
2. Copy the BDD database/s of the choosed game/s to the folder BDD in the install folder. Must be downloaded previously from the page of ESP Translator at Nexus.
3. Execute "ESP Translator.exe".
4. Select the Application Language from the list. This can be changed later in Options/General Options/Application Language.
5. Choose a Main Database for the language to translate. This can be changed later in Options/Main Databases/DataBase.
6. Open the Options Window and then select the Scripts and MCM tab.
7. Choose a translation language in these three settings:
  • MCM Analysis "Default language of translated MCM files"
  • MCM Translation "Default language of translated MCM files"
  • Translation of String files "Default language for String files"
OPTIONAL: To get spell checking in a not available language, read the Hunspell paragraph in the Introduction part above.

UPDATE

The program can be updated by overwriting the old version files for those of the new version, but this process sometimes fails, and it is better to install it separatelly and then copy databases and dictionaries manually from your previous installed version. In general, the folders and files to copy are:

FOLDERS
+ BDD: This folder contains the default databases for games, and the new databases (Main, Custom, OnlyText...) made by you. If newer versions of the Main Databases are uploaded in the mod page, copy first the old version folder and then overwrite them with the downloaded. This way you preserve your own databases and update the default databases.
+ Bases Persos: This folder contains the personal saved databases, those you make when you save an active translation.
+ Dictionaries: This folder contains the spell check dictionaries and synonyms thesaurus. If you installed your owns, you should copy it.
+ Trans Autos: This folder contains the Auto Translation patterns, both the default examples and the own you made.

FILES
+ ESP Translator.settings: These file contains the global setup of the Options window and other tweakeable options in Menus. If you changed them you should copy it.
+ NewDialogConfig.eet: It is supposed to store your settings in the Fallout 4 - Convert for NewDialog Mod tool. So, you may be interested in restore it too.

NOTE: The ESP Translator.settings file is not included in the compressed archive of the program, so the previous personal settings won't be lost when updating, in case you prefer to install the new version over the old one. But, as said above, this method may fail occasionally.
